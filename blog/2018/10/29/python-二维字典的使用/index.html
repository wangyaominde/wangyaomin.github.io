python 二维字典的使用

Posted on 2018-10-29 Edited on 2019-10-14

今天在做项目的过程中需要用到一组三维的结构，eg.‘序号’，‘频率1，频率2’，‘温度’

这时我们可以使用这样的格式：

dict = {‘序号’:{‘温度’:(频率1，频率2)}}
dict={'5':{'30':(10,10)}}
可以得到：
{’30':(10,10)}
查询某一号传感器的某温度的频率信息：
dict['5']['30']
可以得到：
（10，10）
