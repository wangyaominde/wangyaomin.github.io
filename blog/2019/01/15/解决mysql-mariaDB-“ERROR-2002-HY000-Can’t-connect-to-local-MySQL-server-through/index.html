解决mysql/mariaDB “ERROR 2002 (HY000):Can’t connect to local MySQL server through socket ‘/tmp/mysql.sock’ (111)”

Posted on 2019-01-15 Edited on 2019-10-14

搬运&汉化自stackoverflow

解决这个问题需要重启mysql重置密码

1.停止mysql
sudo /etc/init.d/mysql stop
其他版本：
sudo /etc/init.d/mysqld stop

2.进入mysql的安全模式
sudo mysqld_safe --skip-grant-tables &amp;

3.以root用户登录mysql
mysql -uroot

4.选择mysql数据库
use mysql;

5.重置密码
update user set authentication_string=password('mynewpassword') where user='root';

mysql 5.7版本：
update user set authentication_string=password('mynewpassword') where user='root';

6.更新权限
flush privileges;

7.退出
quit

8.重启mysql服务

在CentOS，Fedora ，RHEL中：

9.用新的密码登录
mysql -u root -p
