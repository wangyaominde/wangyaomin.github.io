`userinfo = ‘{“name” : “john”, “gender” : “male”, “age”: 28}’
`需要将上面的变为：
`userdict = {“name” : “john”, “gender” : “male”, “age”: 28}
`
有以下几种方法：

通过 json 来转换

这里有个问题：**数组或对象之中的字符串必须使用双引号，不能使用单引号 **

通过 eval

这个有很严重的安全问题，当用户从config中输入破坏性的代码则会对整个系统存在安全性的威胁

通过 literal_eval_

用这个是比较好的解决方法
